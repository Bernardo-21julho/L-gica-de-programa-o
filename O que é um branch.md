O que é um branch?

Branch uma é um clone independente do código básico de um software que permite aos programadores modificar este sem comprometer a versão original do sistema.



Para que serve?

Branches permitem criar novos recursos, resolver erros e testar modificações de uma maneira controlada e segura.



Aplicações práticas:

Correção de bugs: criar uma branch para corrigir um erro no sistema de login e depois aplicar a correção ao projeto principal.

Novas funcionalidades: desenvolver uma recuperação de senha em uma branch separada antes de adicioná-la ao sistema.

Trabalho em equipe: permitir que vários desenvolvedores trabalhem simultaneamente em diferentes partes do projeto sem interferir no trabalho uns dos outros.
